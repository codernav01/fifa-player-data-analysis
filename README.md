# FIFA Player Data Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the FIFA players dataset to understand player performance, market value, age distribution, nationality trends, and club statistics.  
The goal is to extract meaningful insights from the data and visualize patterns that influence player value and performance.

## Problem Statement
Football clubs invest millions in players. Understanding the factors that influence player performance, wages, and market value is important for clubs, analysts, and scouts.  
This project analyzes FIFA player data to identify trends and insights that help evaluate players.

## Dataset
The dataset contains detailed information about FIFA players including:

- Player Name  
- Age  
- Nationality  
- Club  
- Overall Rating  
- Wage  
- Market Value  
- Position  
- Height & Weight  

Dataset includes thousands of professional football players.

## Key Insights
- Most players are between **22–30 years old**, indicating the prime age for football performance.  
- Players with higher **overall ratings** tend to have higher wages and market values.  
- European clubs dominate the **highest-value players**.  
- Younger players with high ratings often have **higher potential market value**.

## Business Recommendations
- Clubs should invest in **young high-rated players** for long-term value.  
- Scouting departments should focus on **regions producing high-performing players**.  
- Wage structures should align closely with **player performance metrics**.

## Conclusion
This analysis highlights the relationship between **age, performance, wages, and market value** of FIFA players.  
The insights can help clubs, analysts, and scouts make better **data-driven decisions** when evaluating football players.
